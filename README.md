# BELLUS CLICK

`Análise e Desenvolvimento de Sistemas`

`Desenvolvimento de uma Aplicação Interativa`

`2° Semestre`

O Bellus Click consiste em um sistema web voltado para os profissionais da estética. Buscamos auxliar esses profissionais a captar novas clientes, automatizar o processo de agendamento, dentre outras funcionalidades que os auxiliem no dia a dia.
